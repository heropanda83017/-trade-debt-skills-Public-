# 风险→投资条款映射表

## 用途
在投资条款谈判阶段，将尽调中发现的风险转化为具体的条款保护措施。

## 映射总表

| 风险类型 | 条款应对方案 | 优先级 |
|---------|-------------|--------|
| 收入/EBITDA不确定性 | QoE调整、低倍数、earnout、托管、交割后账目对账、价格回调机制 | 必备 |
| 营运资金缺口 | 营运资金peg、购买价调整、现金零债务机制 | 必备 |
| 税务或合规敞口 | 赔偿、托管、专业审查、交割先决条件 | 必备 |
| 客户集中度过高 | 估值折价、客户确认函作为CP、earnout、留存承诺条款 | 建议 |
| 创始人/关键人员风险 | 股权分期归属（vesting）、锁定期、竞业限制、治理权利、关键人员保险 | 建议 |
| IP/产权瑕疵 | 交割先决条件（关闭前整改）、法律意见书、赔偿 | 必备 |
| 诉讼/环保风险 | 专业审查、托管、赔偿、排除、若无法量化则停止交易 | 视严重程度 |
| 重大合同转移/终止风险 | 交易对手同意函（CP）、承诺条款、赔偿、剥离 | 必备 |
| IP归属或许可不明确 | 法律意见书、关闭前完成转让/续期、托管、核心技术无法使用时停止交易 | 必备 |
| 管理层预测不确定性 | 调整基准/下行案例、低估值、earnout、分期投资、里程碑式出资 | 建议 |

## 核心条款审查清单

### 必查项（10项）

1. 证券类型、投资金额、投前/投后估值、持股比例、ESOP处理
2. 资金用途、分期出资、里程碑、先决条件（CP）
3. 董事会构成、观察员席位、保留事项、信息权、检查/审计权
4. 股权转让限制、优先购买权（ROFR）、共同出售权、拖售/随售权、创始人锁定期
5. 清算优先权（liquidation preference）、反稀释保护（anti-dilution）、赎回/回购权、VAM/对赌、股息
6. 创始人承诺、竞业限制、保密、IP归属、雇佣/股权分期归属
7. 陈述与保证、披露函、赔偿上限/门槛/存续期、特别赔偿、托管/预留金
8. 交割机制、重大不利变化（MAC）、签署至交割期间承诺、交割后义务
9. 退出机制：IPO、并购、二次转让、回购路径触发条件和保障
10. 争议解决机制：仲裁vs诉讼、管辖地、适用法律

### 宝钺案特别关注项

| 条款 | 建议方向 | 理由 |
|------|---------|------|
| 估值调整 | 以5亿元为基准价，签署交割后若发现财务差异则调价 | 当前审计报告未出，财务不确定性高 |
| 实控人股权质押 | 增资前完成李素芬/上海芬励尔股权质押至投资方名下 | 防止后续融资稀释/控制权变更 |
| 董事会席位 | 至少1席，重大事项（增资、担保、资产处置）需投资方董事同意 | 军工行业透明度低，需要治理抓手 |
| 优先清算权 | 清算/并购时优先于创始股东获得本金+8%年化收益 | 国有资金保值增值底线要求 |
| 交割先决条件 | 军工资质有效性确认、核心专利无争议、无未披露重大诉讼 | 军工行业特殊合规要求 |
