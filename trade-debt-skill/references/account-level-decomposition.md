# 账户级分解框架（财务核验进阶版）

> 整合自 noahnan-max/financial-red-flag-auditor-skill 的 account-level decomposition 方法论，源于肖星/张新民三表分析体系。

## 核心理念

每个核心科目回答五个问题：
1. **为什么重要？** — 该科目反映什么经营实质
2. **发生了什么变化？** — 金额/占比/同比/环比变化
3. **商业解释能否成立？** — 产品/客户/产能/行业能否解释
4. **会计政策/估计是否影响？** — 确认/计量/列报方式是否合理
5. **在现金流中的位置？** — 现金实际流入/流出是否匹配

## 十大核心科目分析框架

### 1. 货币资金分析

| 问题 | 核查要点 |
|------|---------|
| 余额是否合理？ | 与月均付现成本对比；支持运营的天数 |
| 受限情况？ | 保证金、冻结、共管账户金额 |
| 存贷双高？ | 高现金+高负债同时存在，检查利息收入/平均现金 |
| 是否存在未披露的银行账户？ | 银行流水完整性核查 |

### 2. 应收账款分析

| 问题 | 核查要点 |
|------|---------|
| 增长是否合理？ | 增速 vs 收入增速；剔除新增客户/大单因素 |
| 账龄是否恶化？ | 1年以上/1-2年/2-3年/3年以上分布及同比变化 |
| 坏账计提是否充分？ | 计提比例 vs 同业；实际核销 vs 计提 |
| 期后回款情况？ | 资产负债表日后至尽调日的回款比例 |
| 集中度风险？ | 前5大客户占应收账款比；单一客户信用风险 |

### 3. 存货分析

| 问题 | 核查要点 |
|------|---------|
| 库存结构是否合理？ | 原材料/在产品/产成品/发出商品占比及趋势 |
| 库龄分布？ | 1年以内/1-2年/2年以上占比 |
| 跌价准备是否充分？ | 按库龄+订单覆盖情况计算的跌价 vs 已计提 |
| 周转效率？ | 存货周转天数趋势 vs 同业 |
| 是否有订单覆盖？ | 有订单存货 vs 无订单存货（无订单部分风险更高） |

### 4. 固定资产/在建工程分析

| 问题 | 核查要点 |
|------|---------|
| 产能利用率？ | 设计产能 vs 实际产量；瓶颈工序识别 |
| 在建工程进度？ | 项目明细、预算、完工百分比、长期挂账原因 |
| 折旧政策合理性？ | 折旧年限 vs 资产实际使用寿命；vs 同业 |
| 维护性 vs 扩张性资本开支？ | 区分维持现有产能 vs 新增产能的支出 |

### 5. 收入分析

| 问题 | 核查要点 |
|------|---------|
| 增长驱动力？ | 量价拆分：销量增长 vs 涨价 vs 新品贡献 |
| 客户集中度？ | 前5大/前10大客户占比及变化 |
| 订单覆盖率？ | 在手订单/已签合同 vs 收入预测 |
| 确认时点合规性？ | 收入确认政策（发货/验收/对账/开票） |
| 非经常性收入？ | 一次性大单、政府补助、资产处置等占比 |

### 6. 成本与毛利分析

| 问题 | 核查要点 |
|------|---------|
| 料/工/费结构及趋势？ | 直接材料占比、直接人工占比、制造费用分摊 |
| 议价能力？ | 原材料价格传导能力；是否有长期供货协议 |
| 规模效应？ | 固定成本摊薄 vs 产能利用率的关系 |
| 分产品/分客户毛利？ | 不同产品、不同客户的毛利率差异及合理性 |

### 7. 研发费用分析

| 问题 | 核查要点 |
|------|---------|
| 资本化 vs 费用化？ | 资本化条件是否明确；资本化比例 vs 同业 |
| 研发人员真实性？ | 人数、学历、薪酬 vs 行业；社保+个税比对 |
| 研发投入产出比？ | 研发费用率 vs 新产品收入占比的相关性 |

### 8. 负债与偿债能力分析

| 问题 | 核查要点 |
|------|---------|
| 债务结构？ | 短期 vs 长期；银行借款 vs 债券 vs 融资租赁 |
| 偿债来源？ | 经营现金流能否覆盖利息和到期本金 |
| 续贷风险？ | 银行贷款到期日、续贷条件、是否存在逾期 |
| 表外负债？ | 对外担保、未决诉讼、租赁负债（新租赁准则） |

### 9. 现金流分析

| 问题 | 核查要点 |
|------|---------|
| 造血能力？ | 经营活动现金流是否为正；趋势如何 |
| 净现比（OCF/净利润）？ | 连续年度对比；净利润变为现金的比例 |
| 自由现金流？ | 经营现金流-维持性资本开支 |
| 融资依赖性？ | 经营缺口依靠借债还是股东注资填补 |

### 10. 关联交易分析

| 问题 | 核查要点 |
|------|---------|
| 交易对手是否真实？ | 工商信息查册；是否存在潜在关联关系 |
| 交易定价是否公允？ | 可比非关联方价格/市场价对比 |
| 交易是否必要？ | 是否存在可替代的非关联供应商/客户 |
| 资金往来是否清晰？ | 是否存在长期挂账、循环交易、无商业实质的往来 |

## 应用方法

1. 在完成15维度评分（11.8）和确定性规则筛查（11.9）后，对触发规则/低评分科目进行账户级深度分解
2. 每个科目独立成一个分析卡片
3. 按"数据→商业解释→会计解释→现金流追溯→结论"五步完成每个卡片
