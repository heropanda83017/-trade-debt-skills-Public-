# 预重整尽调研判模式（新增参考文件）

> **来源：** 2026-05-20 Notion #118 #119 + 华源审计 + CITIC电话会三源综合研判会话
> **用途：** 当尽调标的处于预重整/重整状态时使用本模式

---

## 核心模式：三源交叉研判

| 信息源 | 提供内容 | 置信度 |
|--------|---------|--------|
| Notion业务记录（Page #118 #119） | 预重整时间线、债务结构、产业方尽调进展、良品率实测数据 | 高（内部一手） |
| 华源审计审定数据（2026-05-14） | 财务数字（营收2445万/毛利率-166%/现金13万） | 最高（第三方审定） |
| 第三方投行研判（CITIC电话会） | 行业定位、技术评价、估值参考 | 中（外部视角） |

**三源交叉验证检查：**
1. 数字是否一致？ → 不一致时以审计为准
2. 判断是否矛盾？ → 矛盾时拆解各自前提假设
3. 时间节点是否冲突？ → 预重整6月底 vs 审计截止日期6月30日

## 预重整尽调核查清单（新增）

### 🔴 时间窗口类（最高优先级）

| 检查项 | 正常状态 | 红旗信号 |
|--------|---------|---------|
| 预重整预计完成时间 | 有明确deadline | 无具体时间或延后 |
| 产业方接触数量 | ≥2家 | 仅1家或无 |
| 保密协议签署状态 | 已签 | 还在谈判 |
| 尽调进度 | 已启动 | 未启动 |
| 地方政府态度 | 明确支持 | 态度模糊或"无力支持" |

### 🔴 债务结构核实（高优先级）

| 债权人 | 金额 | 性质 | 处理方式 |
|--------|------|------|---------|
| 建设银行 | ？ | 普通债权 | 留债？一次性清偿？ |
| 光大银行 | ？ | 有担保 | 担保物是什么？担保方是谁？ |
| 供应商 | ？ | 普通债权 | 清偿比例？账期？ |

**特别注意：** 光大银行若由地方政府平台公司担保，平台公司代偿后形成对标的公司的债权 → **稀释金瀚股权比例**

## 良品率核查要点

| 指标 | 含义 | 尽调要求 |
|------|------|---------|
| 良品率40% | 盈亏平衡点 | 审计审定数据 |
| 良品率60% | 当前实测数据 | Notion业务记录/实地考察 |
| **趋势判断** | 良品率改善还是恶化？ | 连续3个月数据对比 |

**核查方法：** 
- 实地产线抽查（6个月内的产品批次）
- 客户投诉率/退货率交叉验证
- 与同行业良品率对比（应流约80-90%）

## 产业方尽调优先级判断

**竞争格局四象限：**

```
               有产业方
                 │
     窗口优   │   窗口劣
    ┌──────────┼──────────┐
无产业方       │          │
              │          │
    观望不入  │  快速跟进
              │          
    ──────────────────────────
                 │
              无窗口（已重整）
```

**金瀚策略：**
- 预重整完成前（窗口期）：跟产业方同步尽调，锁定入股价格
- 预重整完成后（竞争期）：估值已锁定，竞争加剧，需更快决策

## 退出路径重新排序（预重整后）

| 优先级 | 路径 | 原因 |
|--------|------|------|
| **① 国央企产业方控股** | 3-4家尽调中，最现实 | 整机厂可承接海外代工，实控人抗性低 |
| ② 财务投资人接盘 | 需等重整完成 | 确定性高于IPO |
| ③ 上市公司收购 | 实控人有抗性 | 路径可行但谈判周期长 |
| ④ 独立IPO | 3-5年 | 短期内不现实 |
