# 制造业/硬科技行业尽调适配器

## 用途
当尽调标的为制造业/硬科技企业时（如宝钺——航空发动机叶片/热端部件制造），使用本适配器替换或补充通用型尽调清单，确保行业特征不被通用框架漏掉。

## 第一步：识别经济引擎

| 引擎类型 | 典型特征 | 核心验证证据 |
|---------|---------|-------------|
| 产品/技术领先型 | 半导体、设备、生物医药、工业软件 | 技术验证、客户认证、IP、技术路线图 |
| 网络/数据/平台型 | 电商、SaaS、互联网消费 | 留存率、用户群、CAC/LTV、数据权 |
| **制造/规模型 ← 宝钺属于此类** | **工业零部件、消费品、材料** | **产能、良率、成本曲线、质量、供应链** |
| 项目/工程型 | EPC、环保工程、系统集成 | 在手订单、验收、应收账款、项目毛利率 |
| 资产/特许经营权型 | 不动产、基础设施、采矿、能源 | 产权、许可、资产质量、现金流合同 |

## 第二步：制造业专属尽调清单

### 2.1 产品与工艺
- [ ] 产品线定义：品类、型号、客户行业、定价机制
- [ ] BOM结构：原材料构成、外购件/自制件比例、关键材料供应商
- [ ] 制造工艺路线：从毛坯到成品的关键工序、专用/通用设备
- [ ] 技术壁垒：工艺know-how、配方、热处理/表面处理等核心环节、专利保护
- [ ] 质量体系：ISO/GJB等体系认证、客户audit记录、废品率/返修率/退货率
- [ ] 产品认证状态：军品/民品各需要什么认证、是否已取得

### 2.2 产能与生产
- [ ] 产能数据：设计产能、实际产能、峰值产能、瓶颈工序
- [ ] 产能利用率：历史12个月的月度利用率、季节性波动
- [ ] 良率/成品率：首检合格率、最终成品率、不良品处理方式
- [ ] 设备状态：关键设备清单、役龄、维护记录、供应商依赖
- [ ] 生产班次：单班/双班/三班、人员配置、加班费成本
- [ ] 外协/外购依赖：哪些工序外包、外包占比、质量控制方式

### 2.3 供应链
- [ ] 原材料供应：关键原材料清单、供应商集中度、采购周期、价格波动
- [ ] 供应商认证：关键供应商是否经过认证、独家供应商风险
- [ ] 库存管理：原材料、在制品、成品库存水平、库存周转率
- [ ] 呆滞料：库龄分布、呆滞料占比、计提跌价准备充分性
- [ ] 物流：运输方式、运费占比、交货周期

### 2.4 客户与订单
- [ ] 订单能见度：在手订单金额、订单周期、订单取消/变更历史
- [ ] 客户认证周期：新客户从接触到首次供货的周期（航空/军工通常1-3年）
- [ ] 客户集中度：前5大客户占比、各客户产品/收入贡献
- [ ] 定价机制：成本加成/目标价/竞标、定价频率、原材料价格传导能力
- [ ] 回款周期：合同信用期、实际回款账龄、逾期情况

### 2.5 财务核验重心
- [ ] 收入确认时点：发货/验收/对账/开票时间点差异
- [ ] 毛利率：按产品/客户/订单的毛利率拆解、毛利率与产能利用率的关系
- [ ] 研发费用：费用化vs资本化比例、研发人员真实性（核查社保+个税）
- [ ] 资产结构：固定资产（设备/厂房）占比、折旧政策合理性
- [ ] 资本开支预测：维护性capex vs 扩张性capex
- [ ] 成本结构：料/工/费占比趋势，规模效应对成本的实际影响

## 第三步：宝钺特别核查项

| 核查项 | 核查要点 | 数据来源 |
|--------|---------|---------|
| 军工资质 | GJB9001C质量体系、武器装备科研生产许可证、保密资质是否齐全有效 | 证书原件 + 军方网站上年度审核记录 |
| 航空发动机叶片 | 客户认证状态（某型号涡扇发动机/燃机）、批量供货vs试制 | 客户采购订单、验收单 |
| 热端部件制造能力 | 定向凝固/单晶铸造工艺成熟度、进口设备依赖度 | 设备清单+维护记录 |
| GT25000燃机 | 批产状态、下游客户（舰船/电站）、在手订单 | 军品合同（隐去机密信息后） |
| 产能瓶颈 | 110MW燃机叶片产能能否支撑2026年8000万产值目标 | 当前产能表+扩产计划 |
| 资金链 | 截至2026.05的现金余额+每月现金消耗率+应收账款回款节奏 | 银行流水+资金预测 |
