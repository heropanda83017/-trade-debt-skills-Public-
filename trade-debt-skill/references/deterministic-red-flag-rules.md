# 确定性红旗规则库（适配非上市公司版）

> 整合自 noahnan-max/financial-red-flag-auditor-skill，保留核心规则逻辑，阈值调整为适配非上市/成长型企业的参数。

## 使用规则

1. 在完成基础数据提取后、15维度评分前运行此规则库
2. 每项规则标记为：已触发 / 未触发 / 数据不可得 / 不可比
3. 同一规则连续2年触发或与其他相关规则同时触发时，升级优先级
4. **重要：** 每项触发仅作为调查线索，不直接等同于负面结论，需通过商业解释测试+会计解释测试+现金流测试后方可下结论

## 严重度映射

| 级别 | 含义 | 典型交易动作 |
|------|------|-------------|
| 🔴 高 | 通常为P1；若触及现金真实性/审计意见/官方欺诈/持续经营/实控人资金占用则为P0 | 暂停推进，启动深度尽调 |
| 🟡 中 | 通常为P2；若持续存在或金额重大或伴有现金流恶化，升级为P1 | 纳入尽调重点核查清单 |
| ⚪ 信息 | 背景信息，不单独扣分 | 记录备查 |

## 规则应用流程

```
运行38条规则 → 标记已触发/未触发/不可得
    → 每项已触发做三测试：
        ① 商业解释测试：行业/客户/产能/产品能否解释？
        ② 会计解释测试：会计政策/估计/确认时点能否解释？
        ③ 现金流测试：现金实际流入流出是否符合收入/利润？
    → 无法解释 → 升级为红旗卡片 → 影响15维度评分
```
