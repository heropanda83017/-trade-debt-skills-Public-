## T-CR 合同审查
```
模板来源：output-templates.md → T-CR v1
审查日期：YYYY-MM-DD
审查性质：签署前审查

一、总体判断（建议签署/修改后签署/不建议签署）
二、协议速览（表格：委托方/受托方/服务/费用/管辖）
三、风险分级审查（🔴红线→🟡高→🟢中→⚪低，每项含条款原文+风险分析+修改建议）
四、国企合规专项（采购程序/国资审批/签署权限/廉洁从业）
五、综合评分与谈判策略（P0不改不签/P1强烈坚持/P2争取修改/P3可接受）
六、后续动作（步骤/动作/责任方/时间节点）
```

## T-IDN 投资研判备忘录
```
模板来源：output-templates.md → T-IDN v1
日期：YYYY-MM-DD
尽调阶段：初步扫描/深入尽调/投决前

一、行业判断（看好/审慎/不看好 + 关键数据）
二、公司判断（强/中/弱 + 3-5条核心发现）
三、财务摘要（营收/毛利率/净利润/现金流/负债率，三年趋势）
四、三法交叉估值（对标交易法+PS对标法+DCF法，各含权重+假设）
五、关键风险矩阵（可能性×影响×缓释措施）
六、投资条款建议（股权质押/优先清算/对赌/董事会等）
七、退出路径评估（并购/IPO/转让/回购，周期+确定性+条件）
八、置信度声明（各判断含高/中/低标注）
```

## T-COL 催收/化债方案评估
```
模板来源：output-templates.md → T-COL v1
催收阶段：预警期/黄牌期/红牌期/法律期/执行期

一、总体判断（优先方案X/暂停/需补充调研）
二、债权概况速览（债务人/金额/账龄/阶段/法律依据）
三、方案比选（每个方案含五维评分：回款确定性/周期/法律保障/成本/合规）
四、推荐路径（A>B>C + 理由 + 关键风险提示）
五、近期动作（优先级/动作/责任方/完成时间）
```

## T-REP 公文/情况汇报
```
模板来源：output-templates.md → T-REP v1
主送单位：

一、基本情况（背景/金额/合同依据/人员）
二、工作进展（一是...二是...三是...，每个含具体动作+结果）
三、存在的问题与风险（已发生/可能发生）

```

## T-FAST 新信息快速研判
```
模板来源：output-templates.md → T-FAST v1
触发事件：[新信息名称]

段1：一句话判断（总体定性）
段2：对策略序列的影响（方案A:↑/↓/—, B:↑/↓/—, C:↑/↓/—）
段3：具体建议（每条含：动作+为什么+预期效果+风险）
段4：需决策事项（表格：事项/建议方向/紧急程度/建议时间）
段5：一句话收尾（闭环）
```
